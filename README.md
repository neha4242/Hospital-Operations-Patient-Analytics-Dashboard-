# 🏥 Hospital Operations Patient Analytics Dashboard

## 📊 Project Overview
This project focuses on analyzing hospital operations and patient data to generate meaningful insights and improve healthcare decision-making. The dashboard provides a comprehensive view of patient demographics, department workload, and operational performance.

---

## 🎯 Project Objectives

- Analyze patient demographics
- Monitor hospital operations
- Identify department workload
- Track patient trends
- Improve hospital efficiency

---

## 📁 Dataset Information

This project uses three datasets:

### Patients Dataset
- Patient ID
- Age
- Gender
- Admission Date
- Discharge Date
- Department
- Patient Type

### Doctors Dataset
- Doctor ID
- Doctor Name
- Department

### Calendar Dataset
- Date
- Month
- Year
- Day Name
- Weekend Indicator

---

## 🔗 Data Model

Relationships Created:
- Patients → Doctors
- Patients → Calendar

Star Schema Data Model used.

---

## 📊 Dashboard Features

- KPI Cards
- Age Distribution
- Patients by Day of Week
- Weekend vs Weekday Analysis
- Department-wise Analysis
- Patient Type Analysis

---

## 📈 Key Metrics

- Total Patients: 4000+
- Average Age: 47
- Total Doctors: 35
- Average Stay: 0.38

---

## 💡 Key Insights

- Most patients aged 40–60
- Weekend admissions higher
- Department workload identified
- OPD patients higher than inpatient
